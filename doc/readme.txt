________________
- Trip to home -
----------------

Assembly game jam 2022 entry

Theme - "The coming back"


Resolution
1024 x 768

Game purpose
------------

You as a player are coming back home from a long trip. Avoid obstacles and crazy things
along the way.


Movement
--------

WASD





Plugins used
------------
Asesprite Wizard
https://godotengine.org/asset-library/asset/713
